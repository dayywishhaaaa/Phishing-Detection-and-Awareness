# Phishing Email Detection & Awareness System

## Overview
This project analyzes phishing email samples to identify phishing indicators, classify email risk, and provide security awareness guidance for end users.

## Objectives
- Analyze phishing email samples
- Inspect email headers
- Identify phishing indicators
- Classify email risk
- Recommend preventive measures
- Document findings in a professional report

## Tools Used
- Kali Linux
- Git & GitHub
- Google Message Header Analyzer
- MXToolbox Email Header Analyzer
- Public phishing email datasets

## Project Structure

Samples
Analysis
